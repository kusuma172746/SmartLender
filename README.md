# 🏦 Smart Lender - Loan Prediction System

## 📌 Project Description
Smart Lender is a Machine Learning-based web application that predicts whether a loan application is likely to be approved or rejected based on applicant details.

## 🚀 Features
- Predicts Loan Approval
- User-Friendly Web Interface
- Machine Learning Model
- Fast and Accurate Prediction

## 🛠️ Technologies Used
- Python
- Flask
- HTML
- CSS
- Scikit-learn
- NumPy
- Pandas

## 📊 Machine Learning Algorithm
- Random Forest Classifier

## 🎯 Project Output
The system predicts whether a loan will be:

✅ Loan Approved
❌ Loan Rejected
